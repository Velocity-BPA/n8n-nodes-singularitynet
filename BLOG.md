- **Blockchain Operations**: Execute transactions and interact with smart contracts on the SingularityNET platform
- **AI Service Access**: Connect to hundreds of decentralized AI services from the SingularityNET marketplace

## Getting Started

Installation is simple through npm:


npm install n8n-nodes-singularitynet


After installation, restart your n8n instance, and you'll find the SingularityNET node available in your node palette. Check out our GitHub repository for detailed documentation, example workflows, and configuration guides: https://github.com/Velocity-BPA/n8n-nodes-singularitynet
